# TG Sistema de Facturación

Trabajo de Grado – Politécnico Colombiano Jaime Isaza Cadavid.

## Descripción
Aplicación de escritorio para apoyar el proceso básico de facturación en una micro/pequeña empresa.

## Alcance (actual)
- Clientes (CRUD)
- Productos/Servicios (CRUD)
- Facturación (ítems, cálculo de totales y exportación a PDF)

Fuera de alcance: reportes avanzados, inventario completo y facturación electrónica.

## Estructura del repositorio
- `docs/`: documentación del Trabajo de Grado (propuesta, requisitos, arquitectura, modelo de datos, bitácoras)
- `src/`: código fuente
- `tests/`: pruebas

## Estado
Documentación v0 ajustada al alcance reducido y en espera de revisión por jurados.
