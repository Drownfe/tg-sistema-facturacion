# Checklist FTG – Práctica (administrativo)

## Por radicar / enviar
- Formato FTG (según modalidad práctica)
- Carta Aval (indicar si hay confidencialidad o no)
- Convenio de práctica
- Acta de compromiso
- Datos de la contraparte (empresa o persona natural)

## Notas
- Si la contraparte aún no está constituida, validar con API si aplica persona natural.
- Mantener versión plantilla (sin datos sensibles) en el repositorio.
